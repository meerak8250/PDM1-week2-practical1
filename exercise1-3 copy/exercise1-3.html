Instructions

Add the setup() and draw() functions to exercise1-3/sketch.js and create a canvas that is at least 500 x 500 pixels. Set the background and fill to any colour you like.

In lecture, we saw how to move a single shape with the mouse by using mouseX and mouseY to set the shape's x and y coordinates. Your animations will often consist of more than one shape that need to be moved as one entity. Therefore, it is important to get comfortable with calculating the coordinates of shapes relative to some other point, such as the mouse location.

Add a 100px wide square to your canvas and set it to move with the mouse. The square should be centred under the mouse--the mouse pointer should be over the centre of the square.

Next, add another 100px wide square and position it so that its bottom right corner touches the top left corner of the original square. Make sure that the new square maintains its position relative to the orginal square when the mouse is moved.

Finally, add a third 100px wide square and position it so that its top left corner touches the bottom right corner of the original square. Again, make sure that the new square maintains its position relative to the other squares when the mouse is moved.

You should have a diagonal line of squares that move in unison when the mouse is moved. Check the expected output below.

When you are done, check your work using check-exercise1-3.html.

Expected output:

You only need to edit sketch.js.

The maths is a lot easier if you draw the squares in CENTER mode.

All x and y coordinates should be calculated relative to mouseX or mouseY.
