Exercise 1.4

Your sketch:

Instructions

Add the setup() and draw() functions to exercise1-4/sketch.js and create a 600 x 600px canvas. Set the background to any colour you like.

Add a rectangle (not a square) to the canvas so that it is in the centre of the canvas. Choose an arbitrary width and height for now.

The goal of this exercise is make the rectangle grow and shrink as the mouse moves as in exercise 1.1 but, instead of growing from the canvas origin, the rectangle will change size based on the mouse's distance from the centre of the canvas. Watch the expected output video below to get a better sense of what you're being asked to do. As you watch, notice that the mouse pointer is always by a corner of the rectangle, but it often changes corner.

You will need to use the system variables mouseX, mouseY, width, and height.

When you are done, check your work using check-exercise1-4.html.

Expected output:

You only need to edit sketch.js.

This exercise requires a little maths. The maths itself is remarkably simple but can be conceptually tricky. Here are some hints:

  1. As the rectangle needs to be centred and its dimensions will grow from the centre, it's a good idea to set the rectangle to draw in CENTER mode. This way, the x and y coordinates of the rectangle will always be the same, no matter where the mouse is.
  2. Focus on calculating just the width of the rectangle. There's no need to tackle both width and height at once. You'll find the maths to be quite similar for both dimensions.
  3. Next, break the problem of calculating the width down into smaller steps. Here is the task: the rectangle should get wider as the mouse moves further from the centre--when the mouse is at the centre, the width should be 0. When the mouse is as far from the centre as it can be, when mouseX is 0 OR 600, the rectangle should be as wide as it can be, which is the width of the screen. You can break the task down as follows:
    • Calculate mouseX's distance from the centre. Your calculation should return the same distance when mouseX is 0 and when mouseX is 600.
    • Figure out how to calculate the rectangle's width from the distance you calculated. This part should be a simple multiplication.
  4. Now you're ready to tackle calculating the height of the rectangle. It should be same process just with different system variables.
