Exercise 1.1

Your sketch:

Instructions

Add the setup() and draw() functions to exercise1-1/sketch.js and create a 600 x 600px canvas. Set the background to any colour you like.

Add a rectangle (not a square) to the canvas. It's top left corner should be at the origin (0, 0) of the canvas and its width and height should grow according to the mouse position--it's bottom right corner should follow the mouse. You will need to use the system variables mouseX and mouseY to create the expected output.

When you are done, check your work using check-exercise1-1.html.

Expected output:

You only need to edit sketch.js.

If you're not sure where to start, begin by copying the sketch starter code from Week 1 Lecture 2 (on the VLE) into sketch.js. Modify the size of the canvas to match the requirements (600 by 600). Then, in draw(), set the background to any colour you like and draw a rectangle using the rect() function. It's location should be 0, 0. For now, choose an arbitrary width and height and make sure you can see it when you run this file in your browser using Live Preview.

To make the bottom right corner of the rectangle follow the mouse, you will need to use mouseX and mouseY to set the width and height of the rectangle.
