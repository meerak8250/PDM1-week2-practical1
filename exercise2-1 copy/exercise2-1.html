Instructions

exercise2-1/sketch.js contains the sliding circle example code shown in lecture. Modify the code to make the circle slide diagonally across the canvas from the top left corner to the bottom right corner. You will need to create a new variable to achieve this output.

When you are done, check your work using check-exercise2-1.html.

Expected output:

You only need to edit sketch.js and you only need to make a few small modifications.

  1. You will need to create a new global variable to store the circle's y position.
  2. Replace the current argument for the circle's y position with your new variable.
  3. Add a line of code modifying the y variable in the same way as the x variable is modified. To move diagonally across a square canvas, the x and y variables should increase at the same rate.
